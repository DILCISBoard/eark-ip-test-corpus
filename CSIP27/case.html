Case CSIP27

E-ARK CSIP:CSIP27
Testable: TRUE

Size of the linked file in bytes.

Rules

  • Rule:1 Packages:1/1
    dmdSec/mdRef/@SIZE is a mandatory attribute if dmdSec/mdRef exist

    ERROR: There is a dmdSec/mdRef element in METS.xml but no @SIZE-attribute. There must be a @SIZE-attribute

    Packages: IP_18000_CSIP27_1 

  • Rule:2 Packages:1/1
    Size of the linked file in bytes as given in METS MUST correspond to the size of the file

    ERROR: There is a mismatch between file size described in METS.xml versus the actual file size

    Packages: IP_18000_CSIP27_2
