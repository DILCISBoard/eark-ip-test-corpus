Case CSIP20

E-ARK CSIP:CSIP20
Testable: TRUE

Status of the metadata. Used to indicate the currency of the package. If used the two values “SUPERSEDED” or “CURRENT” from the vocabulary is used.

Rules

  • Rule:1 Packages:1/1
    There SHOULD be an STATUS attribute for optional element dmdSec in the mets.xml

    WARNING: mets/dmdSec/@STATUS does not exists. It SHOULD be there, it is not a MUST.

    Packages: IP_18000_CSIP20_1 

  • Rule:2 Packages:4/4
    The value of attribute mets/dmd/@STATUS MUST be either "SUPERSEDED" or "CURRENT"

    ERROR: The value of optional attribute mets/dmd/@STATUS is invalid. It MUST be either "SUPERSEDED" or "CURRENT".

    Packages: IP_18000_CSIP20_2  IP_18000_CSIP20_3  IP_18000_CSIP20_4  IP_18000_CSIP20_5
