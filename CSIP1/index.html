Case CSIP1

CSIP:CSIP1
Testable: TRUE Schema: True

mets/@OBJID MUST exist. mets/@OBJID MUST have a non-empty value. mets/@OBJID SHOULD be: a) For the package METS document, @OBJID SHOULD be equal to the name/ID of the package, i.e. the name of the package root folder; b) For a representation level METS document, @OBJID SHOULD be equal to the name/ID of the representation, i.e. the name of the representation folder.
